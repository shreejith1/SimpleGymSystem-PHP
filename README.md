# Gym Management System

A web-based Gym Management System developed using PHP, MySQL, HTML, CSS, and JavaScript to simplify gym administration.

## Features
- Member Registration
- Membership Management
- Attendance Tracking
- Payment Management
- Admin Dashboard
- Secure Login

## Tech Stack
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/shreejith1/SimpleGymSystem-PHP.git
   ```
2. Import the MySQL database.
3. Configure the database connection.
4. Run the project using XAMPP or WAMP.
